# weathermann

Our friend looking up in the clouds for weather report

## Features

- React 16
- Webpack 4
- Babel 7
- Material UI
- Redux
- Standard Version
- Hot Module Replacement

## Installation

```
`git clone https://github.com/prameet-jain/weathermann.git`
cd weathermann
npm install
```

## Run dev

```
npm start

Visit http://localhost:8080/
```

## Production Build

```
npm run build
```
